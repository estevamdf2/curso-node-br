##Curso node.js para iniciantes

#rodando o node
```
$ npm .
```
ou
```
$ npm index.js
```

<p>Para executar um comando especifico</p>
```
$ npm run dev
```

Para o projeto será criado um diretório e um arquivo index.js para cada conceito

<h2>Exemplo: eventos com a classe EventEmitter </h2>
```
$ npm run dev:event
```
